## 🎯 对 Ainovr（墨韵仿写）的适用分析

### 直接能用上的

- **es-toolkit** — lodash 替代品，体积小、性能高，原生 TypeScript。Zustand 管理复杂状态中 debounce、throttle、chunk 等工具函数会频繁用到
- **Biome** — ESLint + Prettier 的 Rust 替代品，一个工具搞定 lint + format，速度快 10-100 倍
- **Rstest** — Jest 的 drop-in 替代，30+ 个 API Route Handler 和 9 个素材提取模块都适合写单元测试
- **React Doctor** — 检测 AI 写的坏 React 代码，检查渲染性能和反模式

### 可以考虑的

- **TanStack AI** — 跨提供商 AI SDK，但项目已自研 SSE 流式客户端，暂不需要
- **React Bits** — 创意 UI 组件库，跟 shadcn/ui 互补，可提升写作用户体验
- **Blazediff** — 视觉回归测试，配合 Vitest/Jest 使用
